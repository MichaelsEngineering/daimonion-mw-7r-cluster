# Security Policy

## Baseline

- Runtime: Python stdlib only.
- Tests: Python stdlib (`unittest`) only.
- CI checks: `compileall` and `unittest`.
- Third-party GitHub Actions pinned to full 40-character SHAs.
- Dependabot updates configured for GitHub Actions.
- CodeQL workflow enabled.
- Build provenance attestation workflow enabled.

## Reporting

Report vulnerabilities through your organization's standard private disclosure channel.
